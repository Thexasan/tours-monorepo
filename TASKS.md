## Ключевые решения / Ограничения

| Решение | Причина |
|---|---|
| Tailwind v4 — CSS-first, токены в globals.css | Нет tailwind.config.ts |
| Email = Nodemailer SMTP | Не менять на Resend или HTTP API |
| roomType хранится в колонке `notes` | Не нужна новая миграция БД |
| NestJS ValidationPipe: `whitelist: true` | Любое неизвестное поле в DTO → 400 ошибка |
| Ken Burns анимация — в globals.css | Не через Tailwind plugin |
| `-mt-16` на hero картинке | Navbar имеет высоту h-16 (sticky) |
